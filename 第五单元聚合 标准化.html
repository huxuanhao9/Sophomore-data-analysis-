def KmeanCut(data,k):
    from sklearn.cluster import KMeans
    kmodel=KMeans(n_clusters=k)
    kmodel.fit(data.values.reshape((len(data),1)))
    c=pd.DataFrame(kmodel.cluster_centers_).sort_values(0)
    w=c.rolling(2).mean().iloc[1:]
    w=[0]+list(w[0])+[data.max()]
    data=pd.cut(data,w)
    return data

三.数据分析

3.1.1

B,C的用电量有波动，A几乎不变 而且2个用户变化有一定周期性，每6个月一次波动，应该是有一定季节性

3.1.2

用户用电量均成类线性增长，且用户C增长最快，量最大，B次之，A最次 B的用电量有波动，其他变化不大

3.2.1

用户21261001一直在进行偷漏电行为，而且集中于9月，警告原因多为电流过负荷和电流不平衡，用电量为800到900之间，线损为0.16左右 用电量变化具有一定波动性，但是趋势为逐渐递减 原因可能是偷漏电行为或者是这几个用户用电量本身的降低

3.2.2

线损率逐年递增，且也有一定周期性，可能随着季节变化波动 线损原因可能是线路老化，人为破化等等

3.2.3

漏电原因具有一定规律性，而且有3种原因发生次数为3次，其他两种分别为2次和4次。基本上原因的时间分布较为均匀，但是有些原因分布比较集中

3.2.4
